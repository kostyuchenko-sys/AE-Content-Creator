# Установка панели AE Content Constructor (для команды)

Пошаговая инструкция. Сделайте один раз — дальше обновления будут автоматически.

---

## Mac

### Шаг 1. Установить Git (если ещё нет)

1. Нажмите **Cmd + Пробел**, введите **Terminal** и откройте программу «Терминал».
2. Вставьте эту строку и нажмите **Enter**:
   ```bash
   git --version
   ```
3. Если появится что-то вроде `git version 2.x.x` — Git уже есть, переходите к шагу 2.
4. Если система предложит установить команды разработчика — нажмите **Установить** и дождитесь окончания.

---

### Шаг 2. Скачать проект на компьютер

1. В том же окне Терминала вставьте по очереди две строки (после каждой нажимайте **Enter**):
   ```bash
   cd ~
   git clone https://github.com/kostyuchenko-sys/AE-Content-Creator.git
   ```
2. Дождитесь окончания (появится снова приглашение вроде `user@mac ~ %`).
3. На компьютере появится папка **AE-Content-Creator** в вашей домашней папке.

---

### Шаг 3. Установить панель в After Effects

1. В Терминале вставьте и нажмите **Enter**:
   ```bash
   cd ~/AE-Content-Creator
   ./scripts/install.sh
   ```
2. Должно появиться сообщение вроде `[deploy] Done` и `[install] Done`.

---

### Шаг 4. Открыть панель в After Effects

1. Полностью закройте After Effects (если был открыт) и откройте снова.
2. В меню выберите: **Window → Extensions (Legacy) → AE Content Constructor**.
3. Панель откроется сбоку.

---

### Шаг 5. Как обновлять панель потом

Когда вам скажут «есть обновление»:

1. Откройте **Терминал**.
2. Вставьте и нажмите **Enter**:
   ```bash
   cd ~/AE-Content-Creator
   git pull
   ```
3. После этого панель в AE уже обновлена (можно перезапустить AE, если что-то не подхватилось).

---

## Windows

### Шаг 1. Установить Git (если ещё нет)

1. Откройте в браузере: **https://git-scm.com/download/win**
2. Скачайте установщик и запустите его.
3. Везде нажимайте **Next** (можно оставить настройки по умолчанию), в конце — **Finish**.
4. После установки в меню Пуск появится **Git Bash** — это окно с чёрным фоном, в нём мы будем вводить команды.

---

### Шаг 2. Скачать проект на компьютер

1. Откройте **Git Bash** (через Пуск или поиск).
2. Вставьте по очереди две строки (после каждой нажимайте **Enter**):
   ```bash
   cd ~
   git clone https://github.com/kostyuchenko-sys/AE-Content-Creator.git
   ```
3. Дождитесь окончания.
4. В папке пользователя (например, `C:\Users\ВашеИмя`) появится папка **AE-Content-Creator**.

---

### Шаг 3. Установить панель в After Effects

**Вариант А — один файл (проще):**

1. Откройте проводник и перейдите в папку:  
   `C:\Users\ВашеИмя\AE-Content-Creator\scripts`  
   (вместо «ВашеИмя» — ваше имя пользователя Windows).
2. Дважды щёлкните по файлу **setup_windows.bat**.
3. Должно открыться чёрное окно, в конце появится что-то вроде `[deploy] Done` и `[install] Done`.

**Вариант Б — через Git Bash:**

1. В **Git Bash** вставьте и нажмите **Enter**:
   ```bash
   cd ~/AE-Content-Creator
   ./scripts/install.bat
   ```

---

### Шаг 4. Открыть панель в After Effects

1. Полностью закройте After Effects (если был открыт) и откройте снова.
2. В меню выберите: **Window → Extensions (Legacy) → AE Content Constructor**.
3. Панель откроется сбоку.

---

### Шаг 5. Как обновлять панель потом

Когда вам скажут «есть обновление»:

1. Откройте **Git Bash**.
2. Вставьте и нажмите **Enter**:
   ```bash
   cd ~/AE-Content-Creator
   git pull
   ```
3. После этого панель в AE уже обновлена (можно перезапустить AE, если что-то не подхватилось).

---

## Если панель не появилась в меню

1. Закройте After Effects и откройте снова.
2. Если не помогло:
   - **Mac:** в Терминале выполните:  
     `cd ~/AE-Content-Creator` и затем `./scripts/deploy.sh`
   - **Windows:** дважды щёлкните по файлу **deploy.bat** в папке `AE-Content-Creator\scripts`, затем снова откройте AE.

---

## Кратко: что вы сделали

- Установили **Git** — программу, которая умеет скачивать и обновлять наш проект.
- Скачали папку **AE-Content-Creator** с панелью и шаблонами.
- Запустили **установщик** — он скопировал панель в папку, откуда её видит After Effects.
- Теперь при **git pull** панель и шаблоны будут обновляться сами.

Шаблоны лежат в папке **projects** внутри AE-Content-Creator. В панели в поле «путь» можно указать эту папку, чтобы выбирать шаблоны оттуда.
